#   01   -   还原系统
##########################################################################
加载失败？ 按照以下步奏操作。

* 从Windows光盘中启动
* 按住SHIFT键，然后按F10
* 输入 "bootsect.exe /nt60 SYS /force" (无引号)
* 重新启动电脑
* 运行上面步骤后输入 "sfc /scannow" (无引号) 以恢复可能已损坏的文件

======
注意：如果命令 "bootsect.exe /nt60 ALL /force" 后还不能正确还原Windows, 请确保您没有添加任何额外的设备 (USB 或其他)，然后再试。

======
注意2:开机后在你的 BIOS 屏幕后按 'R'键。如果你成功了，你会看到你的引导界面. 从中选择"boot without a SLIC" 或 "Fai boot senza SLIC"。

#   07   -   主题 / 调试 / 预激活
##########################################################################
以下参数可使用Re-Loader进行调用，可使用在调试或预激活下。

/ActAuto
同时激活Windows和Office 2016/2013/2010

/ActWindows
激活Windows

/ActOffice
激活Office 2016/2013/2010

/ActOffice14
仅激活Office 2010.

/ActOffice15
仅激活Office 2013.

/ActOffice16
仅激活Office 2016.

/UninstallKMS
卸载KMS

/UninstallOEM
卸载OEM SLIC和安装序列号

/UninstallWPA
卸载AntiWPA(XP)

/NoRestart
在需要时不重新启动

/RestorePoint
在使用Re-Loader之前创建系统还原点

/RemoveWatermark
从Windows中删除水印（需要重新启动）

/DisableTelemetry
禁用微软键盘记录

/SLICert=
手动选择SLIC/证书安装
示例: Re-Loader.exe /SLICert="Dell"

/PK=
手动选择产品密钥
示例: Re-Loader.exe /PK="HP"

/ForceMethod=
如果你想使用另外的KMS服务器地址激活Windows
示例: Re-Loader.exe /Force="KMS"

/Logo=
在同一个文件夹中有文件"OEMLogo" (不含引号)的话，Re-Loader会将Logo.bmp或Logo.reg进行替换

/DumpOEM
另存Windows Vista / 7的Slic、证书和序列号

/LoaderMode=
你可以使用Default, Alternative, Forceful, Old (例如: /LoaderMode="Default")，不同的方法来进行OEM激活

/CheckActivationStatus
检查Windows和Office的激活状态

/ApplicationIntegrityCheck
检查Re-Loader的完整性

/BackupActivation
激活后在桌面上创建一个备份文件夹包含Microsoft产品的激活备份

/FindPID
查找本地计算机上Windows和Office的产品密钥，并进行 PID 检查

/PreserveSLIC
保留系统上已经存在的SLIC



======
注1: 请先准备.NET Framework 4.0 框架
======

======
使用示例 /Logo 1 : Re-Loader.exe /Logo="AutoDetect"
使用示例 /Logo 2 : Re-Loader.exe /Logo="VMWare" (强制安装VMWare的logo)

======
注2: 使用OEM激活将文件中的脚本"SetupComplete.cmd"放入"X:\sources\$OEM$\$$\SETUP\SCRIPTS"中，来让Windows安装时激活

#   08   -   常见问题
##########################################################################
0.  Windows更新：
问：使用该程序后，我可以安装微软系统更新吗？
答：暂时可以。

1.  永久性激活:
问: 这个程序会让微软的产品激活永久吗?
答: 产品可以使用KMS进行每分钟连接KMS激活即可，AntiWPA和OEM方式是永远被激活的。

2.  计划任务
Q:  180天之后不会主动激活怎么办？
A:  保持程序开机自启动，设置激活时间为每1分钟。

3.  报毒
Q:  我的杀毒软件好像神经了？
A:  不是官方下载的，很有可能被修改过。我们网站的下载的话很有可能是误报。

4.  在线/离线
Q:  我需要联网之后再运行这个程序吗？
A:  不需要，但是要确认机器有活动链接。

5.  产品处于未激活状态
Q:  我用过这个程序来激活了，但是不能激活，为何？
A:  可能性太多，先确认产品在不在清单里面，然后很有可能和其他的产品进行冲突的。

6.  语言
Q:  支持哪些语言？
A:  所有的语言都支持！在语言栏目中目前只有几种语言，但是不影响使用。
